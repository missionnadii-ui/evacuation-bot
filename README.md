# 🕊️ Місія Подих Надії — Telegram Bot

Бот для реєстрації евакуйованих осіб.

## Змінні середовища (Environment Variables)

Налаштуйте в Railway:

| Змінна | Опис |
|--------|------|
| `BOT_TOKEN` | Токен від BotFather |
| `GOOGLE_SHEET_ID` | ID таблиці Google Sheets |
| `GOOGLE_PROJECT_ID` | Project ID з Google Cloud |
| `GOOGLE_PRIVATE_KEY_ID` | Private Key ID |
| `GOOGLE_PRIVATE_KEY` | Private Key |
| `GOOGLE_CLIENT_EMAIL` | Client Email |
| `GOOGLE_CLIENT_ID` | Client ID |

## Функції бота

- ➕ Додати евакуйованого — покрокова реєстрація
- 📊 Статистика — кількість осіб сьогодні та загалом
- ℹ️ Допомога — інструкція
